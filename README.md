# WINK BootCamp Team 6 - 프로젝트

## 📌 프로젝트 소개

본 프로젝트는 WINK 부트캠프 6팀이 진행한 웹 프로젝트입니다.  
프론트엔드는 React, 백엔드는 Spring Boot를 기반으로 개발되었으며,  
기본적인 게시판 기능을 중심으로 사용자 간 정보 공유가 가능한 구조로 설계되었습니다.
